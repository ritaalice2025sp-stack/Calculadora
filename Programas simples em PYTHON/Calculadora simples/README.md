# Calculadora Simples em Python

Este é um dos meus projetos iniciais para praticar **lógica de programação** e o uso de **estruturas condicionais** em Python. O objetivo foi criar uma ferramenta funcional que interaja com o usuário via terminal.

## Objetivo do Projeto
Aplicar conhecimentos básicos de Python para resolver um problema cotidiano, focando em:
- Captura e tratamento de dados do usuário (`input`).
- Estrutura de decisão dinâmica (`if/elif/else`).
- Operações matemáticas (aritmética básica e potência).

## Funcionalidades
A calculadora permite realizar 5 tipos de operações:
1. **Soma** (+)
2. **Subtração** (-)
3. **Multiplicação** (*)
4. **Divisão** (/)
5. **Potência** (**)

## Tecnologias Utilizadas
* **Python 3**: Linguagem principal utilizada.
* **VS Code**: Editor de código.

## Aprendizados
Neste projeto, reforcei como converter entradas de texto para números decimais (`float`) e como organizar um menu interativo para que a experiência do usuário seja clara e direta.
